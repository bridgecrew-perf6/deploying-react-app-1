How to Deploy a Create React App to Github

  1. Create a new repository on Github
  2. Install the gh-pages package via the command line: npm install gh-pages --save-dev
  3. In the top level of the package.json, add the 'homepage' details and save: "homepage": "http://{gitname}.github.io/{repository}"
  4. In the 'scripts' located in the package.json, add the following scripts and save:
    • "predeploy": "npm run build",
    • "deploy": "gh-pages -d build"
  5. Set up the repository via the command line using the commands provided from Github when the repository was initially created:
    • git remote add origin https://github.com/{gitname}/deploying-react-app.git
    • git branch -M main
    • git push -u origin main
  6. Add, commit, and push your repository to GitHub using the command line:
    • git add .
    • git commit -m "{comment}"
    • git push
  7. Deploy the site on Github using the 'deploy' script added to the package.json earlier: npm run deploy
